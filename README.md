# AgenticValidator
Use Agentic Program to verify code generated by agent

## Usage

Run the validator from the repo root:

```sh
python agentic_validator.py
```

Optional flags:

```sh
python agentic_validator.py --run-tests --max-line-length 100 --max-function-length 60
```

It scans Python files under the root, reports style and maintainability issues,
and can run pytest when tests are present.
